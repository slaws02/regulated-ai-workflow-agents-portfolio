# Agent 3 — AI Workflow Output Auditor (Demo)

## Purpose
Audits AI outputs for risk, hallucination signals, missing fields, format compliance, confidence alignment, and escalation appropriateness.

## Key Behaviors
- Scores outputs (0–100)
- Flags hallucination risks and unsupported claims
- Detects missing required fields and schema violations
- Produces safer corrected rewrite
- Summarizes issues for QA / governance review
